libparseurl

C library to extract the various elements of a URL.

Summary

libparseurl is a simple set of routines that return a specific peice of information when given a full URL.

Download

Download the source from the following location:

  • libparseurl-src-1.0.tar.gz

Build and Install

libparseurl is easily built and installed. 
  make
  sudo make install

If you don't have sudo installed and configured on your system, log in as root and run the 'make install' command.

A quick example

#include <parseurl.h>

char *host, *path;
int port;

host = url_gethost(url);
path = url_getpath(url);
if (host)  { free(host); }
if (path)  { free(path); }

Parsing the URL

A URL contains multiple peices of information. It contains the host, sometimes the port number, a remote path, and the file to retrieve. Additionally it also contains the protocol used (http, https, ftp, etc).

Any function that returns a string pointer will have allocated memory for it. The developer will be responsible for cleaning up this memory to avoid memory leaks.

Projects or Products that use libparseurl

libparseurl is a rather new library, so it has not been exposed to a lot of developers. However, there are a few other libraries and projects that utilise it.

  • libevhttpget - library to obtain files over http using a libevent event queue.
  • p2net - peer-to-peer web downloading assistance project.
